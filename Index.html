Projects & engineering endeavors

Project 1: Rotary Fatigue Tester

Goal: Build a rotary fatigue tester capable of reaching 1,000,000 continuous strain cycles in ~24 hours.

Tools: Mechanical build + Arduino-based sensing and data logging.

Result: Developed a motion detection system to monitor failures and log cycle counts automatically.

Project 2: Autonomous IR Path-Finding Robot

Goal: Build and program an autonomous path-finding robot that logs uneven terrain using infrared light.

Tools: Microcontroller programming + IR sensing + prototype hardware integration.

Result: Prototyped a smart device pairing workflow for sensor calibration and autonomous movement.
